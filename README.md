# D'Vita System

Sistema de gestión hotelera para **Hospedaje D'Vita**. Incluye módulo de reservas, administración de habitaciones, clientes, empleados, reportes y un chatbot inteligente (DViBot).

---

## Stack Tecnológico

| Capa | Tecnología |
|------|-----------|
| **Frontend** | React + TypeScript + Vite |
| **Backend** | Java + Spring Boot + Maven |
| **Base de datos** | MySQL |
| **Chatbot API** | Python + FastAPI |
| **IA Local** | Ollama |
| **Estilos** | Tailwind CSS |

## Chatbot DViBot

Asistente virtual integrado en el frontend que permite a los huéspedes:

- Consultar disponibilidad de habitaciones
- Ver precios y servicios
- Crear, cancelar y consultar reservas
- Obtener información de contacto y ubicación
- Navegación por menús interactivos
